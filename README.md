# Data-Analytics-power-BI-python-
Customer Behavior Data Analyst Portfolio Project

This project demonstrates an end-to-end data analytics workflow by transforming raw customer behavior data into actionable business insights. The dataset was cleaned and analyzed using Python, stored in PostgreSQL, explored through SQL queries, and visualized in an interactive Power BI dashboard. The project highlights customer segments, purchasing patterns, loyalty trends, and business performance to support data-driven decision-making.
